# UMD Region Changer
##### Presented by ARK-4 Team

1. After compiling the files will be in `dist`

2. After copying `UMD_Region_Changer` to `ms0:/PSP/GAME` run it.

3. Menu shows which region you want to your UMD Drive to mimic.

4. Select and enjoy!
